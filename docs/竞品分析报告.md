# AI 智能招投标平台竞品分析报告

> 编制日期：2026-05-28
> 项目定位：面向乙方（投标方）的投标文件智能预审平台，核心能力为招标文件拆解、废标风险预检

---

## 一、市场背景

### 1.1 政策驱动

2026年2月，国家发改委等八部委联合发布《关于加快招标投标领域人工智能推广应用的实施意见》（发改法规〔2026〕195号），提出：

- 到2026年底，实现**招标文件检测、智能辅助评标**等关键功能落地
- 明确20个AI应用场景，覆盖招标→投标→开标→评标→定标→合同管理全流程
- 各省市（广东、海南、雄安等）已开始落地试点

### 1.2 痛点数据

- 2025年因标书格式错误、条款响应遗漏等低级失误造成的**废标占比高达37%**
- 传统标书制作平均耗时30天，AI工具可实现分钟级生成，效率提升80%+
- 企业采购评审文件写作效率提升50%，评审效率提升70%（人民网报道）

### 1.3 市场分层

| 层级 | 特征 | 代表产品 |
|------|------|---------|
| 基础型 | 模板拼接、格式调整，适合简单商务标 | 小羊标书、镖行AI标书等 |
| 专业级 | 全流程智能管控（解析→生成→检查→签章） | 数商云、标桥、北京筑龙 |
| 平台级 | 全流程交易平台+AI能力 | 广联达、国能e招、e招智胜 |

## 六、核心发现与差异化定位

### 6.1 市场空白

1. **乙方视角的废标预审**：现有产品中，北京筑龙、广联达、来也科技均面向甲方（评标方），数商云、标桥等偏向标书生成。**专门面向乙方进行投标前废标预检的产品极为稀缺**。
2. **多Agent协作架构**：除本项目使用Mastra多Agent系统外，仅有来也科技采用智能体架构，其余产品仍为传统流程式处理。
3. **问题精准定位**：本项目支持页/块级坐标定位，多数竞品仅提供文本级反馈。

### 6.2 最值得借鉴的产品

| 借鉴方向 | 推荐产品 | 理由 |
|---------|---------|------|
| **废标检测规则库** | 北京筑龙（智能清标） | 客观项批量核查模式成熟，应标点自动检查+高亮提示 |
| **合规检测维度** | e招智胜（广州） | 137个检测点覆盖，95%准确率，检测维度最全面 |
| **标书检查功能** | 标桥 | 关键指标100%匹配检测，与本项目定位最接近 |
| **智能体架构** | 来也科技 | OCR+NLP+LLM全链路，智能体自主编排 |
| **逐条合规审查** | Specgen.ai | "逐条对标"的合规矩阵模式 |
| **开源参考** | OpenBidKit_Yibiao | 废标项检查实现思路可直接参考 |
