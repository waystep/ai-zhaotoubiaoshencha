# v0.0.2 Phase 1: Platform Foundation — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 建立 v0.0.2 平台基础层——认证体系升级 + 模型管理系统，为后续知识库、智能体、业务流程提供地基。

**Architecture:** 在 v0.1.0 现有 NextAuth v5 + Drizzle ORM 基础上，扩展认证方式（手机号+验证码、飞书/微信 OAuth、企业 SSO），新增模型注册表和智能体-模型分配表。所有新表通过 Drizzle migration 管理。

**Tech Stack:** Next.js 15, NextAuth v5 (beta), Drizzle ORM, PostgreSQL, Tailwind CSS, shadcn/ui

**Spec:** `docs/superpowers/specs/2026-05-28-v0.0.2-platform-design.md` Section 十一 + Section 五

```typescript
export interface SMSProvider {
  sendCode(phone: string, code: string, purpose: string): Promise<{ success: boolean; requestId?: string }>;
}

export class SMSService {
  private provider: SMSProvider;
  private codeLength = 6;
  private codeExpiryMinutes = 5;
  private maxSendPerHour = 5; // 同一号码每小时最多发送 5 次

  constructor(provider: SMSProvider) {
    this.provider = provider;
  }

  generateCode(): string {
    return Math.random().toString().slice(2, 2 + this.codeLength);
  }

  getExpiryMinutes(): number {
    return this.codeExpiryMinutes;
  }
}

// 工厂函数：根据环境变量选择 SMS 提供商
export function createSMSService(): SMSService {
  const provider = process.env.SMS_PROVIDER === "tencent"
    ? new TencentSMSProvider()
    : new AliyunSMSProvider();
  return new SMSService(provider);
}
```

import { OAuthProvider } from "next-auth/providers";

export function FeishuProvider(): OAuthProvider {
  return {
    id: "feishu",
    name: "飞书",
    type: "oauth",
    authorization: {
      url: "https://open.feishu.cn/open-apis/authen/v1/authorize",
      params: {
        app_id: process.env.FEISHU_APP_ID,
        redirect_uri: process.env.FEISHU_REDIRECT_URI,
        state: "feishu",
      },
    },
    token: {
      url: "https://open.feishu.cn/open-apis/authen/v1/oidc/access_token",
      async request(context) {
        // 飞书 token 交换逻辑
        // 用 app_access_token + user_access_code 换取 user_access_token
      },
    },
    userinfo: {
      url: "https://open.feishu.cn/open-apis/authen/v1/user_info",
      async request({ tokens }) {
        // 获取飞书用户信息
      },
    },
    profile(profile) {
      return {
        id: profile.open_id,
        name: profile.name,
        email: profile.email,
        image: profile.avatar_url,
      };
    },
  };
}
