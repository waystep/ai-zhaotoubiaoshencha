# 智能投标预审智能体 数据库设计文档

## 1. 数据库概述

### 1.1 技术选型

| 项目 | 技术 | 说明 |
|------|------|------|
| **数据库** | PostgreSQL | 主数据存储 |
| **ORM** | Drizzle ORM | TypeScript 类型安全 |
| **迁移工具** | Drizzle Kit | Schema 管理 |
| **连接方式** | `postgres` 库 | 连接池支持 |

### 1.2 配置信息

```typescript
// drizzle.config.ts
{
  schema: "./src/lib/db/schema.ts",
  out: "./drizzle",
  dialect: "postgresql",
  dbCredentials: {
    url: process.env.DATABASE_URL || "postgresql://localhost:5432/smart_tender_review"
  }
}
```

### 1.3 数据库连接

```typescript
// src/lib/db/client.ts
import { drizzle } from "drizzle-orm/postgres-js";
import postgres from "postgres";
import * as schema from "./schema";

const client = postgres(process.env.DATABASE_URL!);
export const db = drizzle(client, { schema });
```

## 7. 数据隔离策略

### 7.1 组织级数据隔离

所有业务数据通过 `orgId` 实现组织级隔离：

- 项目：`tenderProjects.orgId`
- 文档：通过 `documents.projectId` → `tenderProjects.orgId`
- 报告：通过 `reviewReports.projectId` → `tenderProjects.orgId`

### 7.2 权限验证

API 层应验证用户所属组织，确保数据访问权限：

```typescript
// 示例：获取用户组织
const membership = await db.query.organizationMembers.findFirst({
  where: eq(organizationMembers.userId, session.user.id),
});

// 查询时过滤 orgId
const projects = await db.query.tenderProjects.findMany({
  where: eq(tenderProjects.orgId, membership.orgId),
});
```
